# Imaginary Image Repository

A repository with public and private images. It automatically recognizes what you have on the image you upload and tags the photos appropriately. Then, you can easily search for "that photo from wedding" across your 10000 photo collection.

### Technologies Used
- React on the front-end
- Firebase on the back-end for authentication, storage and to replace a server. This applciation is server-less and uses cloud functions instead of server endpoints.
- Google Cloud API. A pre-trained ML model to tag the photos.

### How to use

1. Go [here](https://image-repository-c1030.web.app/) and login

![Login Screen](https://www.linkpicture.com/q/1_751.png)


2. On the main screen, you can toggle between Public and  mode Private in the app bar. Also, notice the tags under each photo. To search for the desired tag, enter the text in the search bar. To upload your picture, click the icon in the top left corner to see the next screeen,

![Main Screen](https://www.linkpicture.com/q/2_476.png)


3. Here you can upload pictures in bulk and make them private to you or public (accessible to anyone) by toggling the checkbox. When you are done selecting the images, press the button in the top right corner to come back to screeen 2.

![Main Screen](https://www.linkpicture.com/q/3_338.png)
